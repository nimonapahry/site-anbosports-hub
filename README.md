# site-anbosports-hub

## 项目简介

site-anbosports-hub 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库主要用于存放各类静态页面资源，不针对任何特定域名或网站，仅作为页面文件的集中管理空间。

## 目录结构

```
site-anbosports-hub/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── docs/           # 页面相关文档说明
└── README.md       # 本文件
```

- **pages/**：包含所有归档的 HTML 页面，每个页面以独立文件夹或文件形式存放。
- **assets/**：页面所需的公共资源，如样式表、脚本和图像。
- **docs/**：页面的使用说明、变更记录等文档。

## 页面归档说明

本仓库中的页面文件均为静态 HTML，可直接在浏览器中打开查看。页面内容可能包含演示、测试或存档用途，不保证实时更新或持续维护。

如需添加新页面，请将完整 HTML 文件放入 `pages/` 目录，并确保引用资源路径正确。
